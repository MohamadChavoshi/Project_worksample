# Simple Online Shop

A modern and lightweight online shop project built with React and Vite using JSX.
This project focuses on clean architecture, maintainable code structure, and scalable frontend development practices.

The application currently includes basic shopping features and dynamic product filtering using URL search parameters.
The project is still under development and is not production-ready yet.

Future updates will include a complete admin panel and content management/control center for managing products, categories, and store data.

---

## 🚀 Features

* ⚛️ Built with React + Vite
* 📦 JSX-based component structure
* 🔍 Product filtering using URL Search Params
* 🧹 Clean Code principles and maintainable architecture
* 📱 Responsive UI
* ⚡ Fast development experience with Vite
* 🛠️ Easy to scale and extend
